Binary downloads are available for Linux, Windows and Mac (Darwin).

:::note
Downloads vary between 20 and 120 MiB in size. When extracted, each download contains a directory with the following content:

<FileTree>

- edgeiq or edgeiq.exe
- README.md
- LICENSE

</FileTree>
:::

Refer to the [installation guide](/edge_iq/install/overview) for installation & configuration guides.
