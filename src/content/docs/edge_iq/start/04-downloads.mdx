## License Registration

Before downloading EdgeIQ, you'll need to register for a free license. Visit the [EdgeIQ License Registration](https://edge-iq-services.behavure.ai/l/edgeiq-free-license) page to get started. After registration, you'll receive your license key which will be required during the installation process.

Refer to the [installation guide](/edge_iq/install/overview) for installation & configuration guides.

## Latest stable release (v1.1.3)

:::caution
**macOS Users**: We're actively working code signing for the macOS binary. After downloading EdgeIQ for macOS, run the following command to remove the quarantine attribute:

```
xattr -d com.apple.quarantine ./edgeiq
```

:::
