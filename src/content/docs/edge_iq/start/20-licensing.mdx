Lyftdata server requires a valid license.

To obtain a free evaluation license via email, please [click here](http://services.lyftdata.com/l/eval-license).

:::note
Evaluation licenses are valid for 45 days, and accommodates up to 500 GiB of data processing per day, on up to 15 workers.
:::

Please [contact us](https://lyftdata.com/contact/) for additional licensing options and related pricing.

## Configuring a server license

To configure a license on a Lyftdata server, follow these steps:

<Steps>

1. Login in to the UI
2. Open the navigation menu on the top left corner
3. Select _manage_ -> _licensing_
4. Click the _active license_ button
5. Paste the license key and click the _activate_ button.

</Steps>
