**Knowledge objects** are user-created query building blocks in Scuba. These include [metrics](../metric), [actor properties](../actor-property), manual [event properties](../event-property), [flows](../flow), [flow properties](../flow-property), and [segments](../segment). In many communications, "knowledge object" is shortened to "knob."
