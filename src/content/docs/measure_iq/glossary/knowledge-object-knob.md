---

**Knowledge objects** are user-created query building blocks in Measure IQ. These include [metrics](metric), [actor properties](actor-property), manual [event properties](event-property), [flows](flow), [flow properties](flow-property), and [segments](segment). In many contexts, knowledge objects are referred to as "knobs."
