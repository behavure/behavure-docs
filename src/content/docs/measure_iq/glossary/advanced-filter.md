---
title: Advanced Filter 
description: Definition & use of Advanced Filter 
---
You can use **Advanced Filters** in the [Query Builder](../query-builder) to compare anything in Scuba. You can create filters to compare not only columns to values, but also to compare columns to columns and metrics to metrics (or columns to metrics).
